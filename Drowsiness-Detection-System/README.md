# Drowsiness Detection System

A computer vision and artificial intelligence system designed to detect driver drowsiness in real time.

## Overview

This project uses facial landmark detection and a trained convolutional neural network (CNN) to monitor signs of driver fatigue and trigger an alert when drowsiness is detected.

## Technologies

* Python
* OpenCV
* MediaPipe
* TensorFlow
* NumPy

## Features

* Real-time face tracking
* Facial landmark analysis
* CNN-based drowsiness classification
* Audible alert system
* Live webcam monitoring

## Applications

* Driver safety systems
* Human monitoring systems
* Computer vision research

## Dataset and Model

The original dataset and trained model are not included in this repository.

The dataset contains images of individuals collected during the development of the project and is not publicly distributed for privacy reasons.
